## Connect Frontend (React + Vite)

This folder contains the React client for **Connect - Global Real-time Messaging App**.

For full project documentation (features, API, setup, deployment), see the root [`README.md`](../README.md).

### Local commands

- `npm install` – install dependencies
- `npm run dev` – start Vite dev server (default `http://localhost:5173`)
- `npm run build` – production build
- `npm run preview` – preview production build
